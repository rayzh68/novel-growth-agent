## 增长系统职责

目标：
发现值得投入资源的小说项目，并负责后续增长策略。

负责：

1. 中文原始小说商业价值评估

输出：
- Commercial Promise
- Creative Potential
- Binge Potential
- Test50 Value
- Investment Attractiveness

2. 给出决策建议

输出：
- SKIP
- TEST_50_CHAPTERS
- HIGH_PRIORITY

3. 多书排序和优先级

例如：
Book A
Book B
Book C

输出优先级。

4. 根据生产工厂产出的前50章进行：

- SEO分析
- 投放素材生成
- 广告角度提取
- Landing Page策略
- 投放计划（Facebook、TikTok等）

5. 根据市场反馈更新优先级
